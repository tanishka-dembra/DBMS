# User Guide

## Recruiter Flow

1. Register with recruiter details and company name.
2. Sign in with the registered email and password.
3. Complete the company profile before submitting forms.
4. Create a JNF or INF as a draft.
5. Fill profile, salary or stipend, eligibility, selection process, and declaration details.
6. Upload supporting PDFs or images where required.
7. Submit the form for admin review.
8. Track status from the dashboard or submissions list.
9. Read notifications for approvals, rejections, or edit requests.
10. If a form is opened for edit, update it and submit again.

## Admin Flow

1. Sign in through the admin login page.
2. Review dashboard metrics for companies and submissions.
3. Open JNF or INF review queues.
4. Inspect full submission details and history.
5. Approve, reject, or request edits with clear remarks.
6. Manage company profile data from the company management area.
7. Use notifications to track new registrations and review outcomes.

## Password Reset

Use the forgot-password action from login. The reset link expires after 30 minutes and can be used once.
